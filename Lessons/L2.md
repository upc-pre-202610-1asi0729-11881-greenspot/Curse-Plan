## Lección 2 — Variables y Tipos de Datos

### ¿Qué es una variable?

Piensa en una variable como una **caja con etiqueta**.

Imagina que tienes una caja en tu cuarto con una etiqueta que dice *"dinero para el recreo"*. Adentro puedes poner S/5, S/10, o lo que sea. Si el próximo día cambias el monto, el nombre de la caja sigue siendo el mismo, pero lo que hay adentro cambia.

En programación pasa exactamente lo mismo:

```java
int vida = 100;
```

Aquí:
- `int` → es el tipo de caja (solo caben números enteros).
- `vida` → es la etiqueta de la caja.
- `100` → es lo que hay adentro.
  Si el personaje recibe daño, puedes cambiar el contenido:

```java
vida = 80;
```

La caja sigue llamándose `vida`, pero ahora tiene 80 adentro.
 
---

### ¿Para qué sirven las variables?

Sin variables, un programa no puede recordar nada. Sería como jugar un videojuego que cada vez que recibes daño olvida cuánta vida tenías.

Las variables le dan **memoria** al programa. Guardan información que puede usarse, compararse o cambiarse más adelante.

En nuestro videojuego necesitamos recordar, por ejemplo:
- El nombre del personaje.
- Cuánta vida le queda.
- Cuánto daño hace con cada ataque.
  Para eso usamos variables.

---

### Tipos de datos en Java

No todas las cajas son iguales. Dependiendo de qué quieras guardar, necesitas un tipo diferente. En Java los principales son:

#### `int` — Números enteros

Para guardar números sin decimales: edad, vida, puntos, cantidad de monedas, etc.

```java
int vida = 100;
int ataque = 25;
int edad = 15;
```

**Ejemplo de la vida real:** el marcador de un partido de fútbol. No puedes tener 2.5 goles. Es siempre 0, 1, 2, 3...
 
---

#### `double` — Números con decimales

Para guardar números que pueden tener punto decimal: precios, promedios, temperaturas, etc.

```java
double precio = 9.99;
double promedio = 14.7;
double temperatura = 36.5;
```

**Ejemplo de la vida real:** el precio de algo en una tienda. S/9.99 no es lo mismo que S/10.
 
---

#### `String` — Texto

Para guardar palabras, frases, nombres, cualquier cosa escrita. Siempre va entre comillas `"..."`.

```java
String nombre = "Aragorn";
String ciudad = "Lima";
String mensaje = "¡Bienvenido al juego!";
```

**Ejemplo de la vida real:** tu nombre de usuario en una red social. Es texto, no un número.

> ⚠️ Importante: `String` se escribe con **S mayúscula**. Es el único tipo básico que se escribe así en Java. Si lo escribes en minúscula, el programa no lo entiende.
 
---

#### `boolean` — Verdadero o Falso

Solo puede tener dos valores: `true` (verdadero) o `false` (falso). Nada más.

```java
boolean estaVivo = true;
boolean tieneEscudo = false;
boolean juegoTerminado = false;
```

**Ejemplo de la vida real:** una lámpara. Solo puede estar encendida o apagada. No hay punto medio.

En el videojuego usamos `boolean` para saber si un personaje todavía está en pie.
 
---

### Comparación rápida de los 4 tipos

| Tipo | ¿Qué guarda? | Ejemplo |
|---|---|---|
| `int` | Números enteros | `int vida = 100;` |
| `double` | Números decimales | `double precio = 4.50;` |
| `String` | Texto | `String nombre = "Kiara";` |
| `boolean` | Verdadero o falso | `boolean vivo = true;` |
 
---

### Las variables de nuestro personaje

Vamos a aplicar todo lo que aprendiste. Así se ven las variables del personaje principal de nuestro videojuego:

```java
String nombre   = "Aragorn";   // nombre del héroe
int vida        = 100;         // puntos de vida iniciales
int ataque      = 25;          // daño que hace por turno
boolean estaVivo = true;       // ¿sigue en pie?
```

Y las del villano:

```java
String nombreVillano   = "Sauron";
int vidaVillano        = 120;
int ataqueVillano      = 30;
boolean villanoVivo    = true;
```

Con esto, el programa ya "sabe" quiénes son los dos personajes. En las próximas lecciones les vamos a agregar acciones.

### Mostrar variables en pantalla

Para ver el valor de una variable, usamos `System.out.println()`:

```java
String nombre = "Aragorn";
int vida = 100;
 
System.out.println("Personaje: " + nombre);
System.out.println("Vida: " + vida);
```

El `+` une el texto con la variable. En pantalla verías:

```
Personaje: Aragorn
Vida: 100
```

**Pruébalo tú mismo →** [Abrir en JDoodle](https://www.jdoodle.com/online-java-compiler), escribe el código y ejecútalo. Luego cambia el nombre y el valor de vida y vuelve a correrlo. ¡Eso es programar!

### Errores comunes al usar variables (y cómo evitarlos)

Todos los programadores cometen estos errores al principio. Conocerlos de antemano te ahorra tiempo:

| ❌ Error común | ✅ Cómo se hace bien |
|---|---|
| `string nombre = "Ana";` | `String nombre = "Ana";` → S mayúscula |
| `int vida = "100";` | `int vida = 100;` → sin comillas para números |
| Usar la variable antes de declararla | Siempre declara primero, usa después |
| `int promedio = 14.7;` | `double promedio = 14.7;` → usa `double` para decimales |
 
---

### Resumen de la Lección 2

- Una **variable** es una caja con nombre que guarda un dato.
- Antes de usar una variable hay que **declararla** con su tipo y nombre.
- Los 4 tipos básicos son `int`, `double`, `String` y `boolean`.
- Puedes mostrar variables en pantalla con `System.out.println()`.
- En el videojuego, las variables guardan el nombre, la vida y el ataque de cada personaje.
