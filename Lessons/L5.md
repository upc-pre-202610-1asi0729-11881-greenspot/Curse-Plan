## Lección 5 — Fundamentos de POO y el Juego Completo (15 min)

Este es el momento. Todo lo que aprendiste hasta ahora — variables, condiciones, bucles, métodos — fue preparación para esto. Hoy lo unimos todo en la herramienta más poderosa de Java: **las clases y los objetos**.

---

### ¿Qué es una clase y qué es un objeto?

Piensen en un molde de galletas. El molde define la forma — dice que toda galleta tiene esta figura, este tamaño. Pero el molde no es una galleta. Las galletas son lo que produces con ese molde.

* **La Clase (El Molde):** En programación, la clase es el molde. En nuestro juego, vamos a crear una clase llamada `Personaje`. Esa clase define que todo personaje tiene un nombre, una vida y un ataque.
* **El Objeto (La Galleta):** Es algo real, concreto, que creaste a partir de ese molde. De esa clase vamos a crear dos objetos: el héroe y el enemigo. Cada uno con sus propios valores, pero ambos siguiendo el mismo molde.

---

### Atributos y métodos dentro de una clase

Una clase tiene dos cosas fundamentales para mantener el código organizado, limpio y reutilizable:

1. **Atributos:** Son las características — los datos que definen al objeto. Para nuestro `Personaje`: `nombre`, `vida` y `ataque`.
2. **Métodos:** Son las acciones — lo que el objeto puede hacer. Para nuestro `Personaje`: `atacar()` a otro personaje y `mostrarEstado()`.

---

### Crear nuestra primera clase: el Personaje del videojuego

Vamos al código. Así se ve nuestra clase `Personaje` explicada línea por línea.

Miren el **constructor** — ese método especial que tiene el mismo nombre que la clase. Es lo que usamos para crear un personaje nuevo con sus valores. La palabra `this` significa "este objeto" — es la forma en que la clase se refiere a sí misma.

```java
class Personaje {
    String nombre;
    int vida;
    int vidaMax;
    int ataque;
    int pociones;
    Random rand = new Random();

    // Constructor
    Personaje(String nombre, int vida, int ataque, int pociones) {
        this.nombre = nombre;
        this.vida = vida;
        this.vidaMax = vida;
        this.ataque = ataque;
        this.pociones = pociones;
    }

    // Método: atacar
    void atacar(Personaje defensor) {
        int dano = this.ataque + rand.nextInt(6);
        defensor.vida = Math.max(0, defensor.vida - dano);
        System.out.println(this.nombre + " ataca a " + defensor.nombre + " por " + dano + " puntos!");
        System.out.println(defensor.nombre + " le queda " + defensor.vida + " de vida.");
    }

    // Método: mostrarEstado
    void mostrarEstado() {
        System.out.println(this.nombre + " | Vida: " + this.vida + "/" + this.vidaMax +
                           " | Pociones: " + this.pociones);
    }

    // Método: usarPocion
    void usarPocion() {
        if (this.pociones > 0) {
            int curacion = 30;
            this.vida = Math.min(this.vida + curacion, this.vidaMax);
            this.pociones--;
            System.out.println(this.nombre + " usa una poción y recupera " + curacion + " de vida!");
            System.out.println("Vida actual: " + this.vida + " | Pociones restantes: " + this.pociones);
        } else {
            System.out.println("¡No te quedan pociones!");
        }
    }
}

```

### Instanciar objetos y usarlos en el programa

Ahora que tenemos el molde, vamos a crear los dos personajes del juego. Esto se llama **instanciar un objeto** — crear una galleta a partir del molde.

Con la palabra reservada `new Personaje()` le decimos a Java: crea un objeto real usando esta clase.

```java
Personaje heroe   = new Personaje("Guerrero", 100, 15);
Personaje enemigo = new Personaje("Dragón", 120, 12);
```

El héroe tiene 100 de vida y 15 de ataque. El dragón tiene 120 de vida y 12 de ataque. Dos objetos distintos, mismo molde.

### Construimos: la clase Personaje, dos instancias y el combate final — ¡el juego queda completo!

Ahora unimos todo. Aquí está el programa completo interactivo. Escríbanlo en sus computadoras, córranlo, ingresen el nombre de su personaje y vean el combate desarrollarse turno a turno usando un bucle `while` hasta que uno se quede sin vida.

```java
import java.util.Scanner;

public class Leccion5_Simple {

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);

        System.out.println("╔══════════════════════════════╗");
        System.out.println("║       COMBATE RPG - JAVA      ║");
        System.out.println("╚══════════════════════════════╝\n");

        System.out.print("¿Cómo se llama tu personaje? ");
        String nombre = scanner.nextLine();

        System.out.println("\nElige tu clase:");
        System.out.println("  1. Guerrero — 120 vida | 18 ataque | 3 pociones");
        System.out.println("  2. Mago     —  80 vida | 22 ataque | 2 pociones");
        System.out.print("Elige (1/2): ");
        String clase = scanner.nextLine();

        // Creamos (Instanciamos) nuestros dos objetos en lugar de usar variables sueltas
        Personaje heroe;
        if (clase.equals("2")) {
            heroe = new Personaje(nombre, 80, 22, 2);
        } else {
            heroe = new Personaje(nombre, 120, 18, 3);
        }

        Personaje enemigo = new Personaje("Dragón Oscuro", 150, 14, 0);

        System.out.println("\n=== COMBATIENTES ===");
        heroe.mostrarEstado();
        System.out.println("Dragón Oscuro | Vida: " + enemigo.vida);
        System.out.println("\n¡Que comience el combate!\n");

        int turno = 1;

        // Bucle usando directamente la vida de los objetos
        while (heroe.vida > 0 && enemigo.vida > 0) {

            System.out.println("════════════════════════════");
            System.out.println("  Turno " + turno);
            heroe.mostrarEstado();
            System.out.println("  Vida enemigo: " + enemigo.vida);
            System.out.println("════════════════════════════");
            System.out.println("¿Qué haces?");
            System.out.println("  1. Atacar");
            System.out.println("  2. Usar poción");
            System.out.println("  3. Retirarse");
            System.out.print("Elige: ");
            String opcion = scanner.nextLine();
            System.out.println();

            if (opcion.equals("1")) {
                heroe.atacar(enemigo);
            } else if (opcion.equals("2")) {
                heroe.usarPocion();
            } else if (opcion.equals("3")) {
                System.out.println(heroe.nombre + " se retira. ¡Hasta la próxima!");
                scanner.close();
                return;
            } else {
                System.out.println("Opción no válida. ¡Pierdes el turno!");
            }

            if (enemigo.vida > 0) {
                System.out.println();
                enemigo.atacar(heroe);
            }

            System.out.println();
            turno++;
        }

        System.out.println("=== FIN DEL COMBATE ===");
        if (heroe.vida > 0) {
            System.out.println("¡VICTORIA! " + heroe.nombre + " ha derrotado al Dragón Oscuro!");
        } else {
            System.out.println("DERROTA. ¡Inténtalo de nuevo!");
        }

        scanner.close();
    }
}
```
