## Lección 1 — ¿Qué es la Programación y qué es Java?

### ¿Qué es programar?

Imagina que tienes un amigo robot que puede hacer *cualquier cosa* que le pidas, pero con una condición: **tiene que entender exactamente lo que le dices**. No puede adivinar, no puede suponer. Si le dices "prepara algo de comer", se queda paralizado porque no sabe qué, cuánto, ni cómo.

Pero si le dices:

1. Abre el refrigerador.
2. Saca 2 huevos.
3. Enciende el fuego a temperatura media.
4. Espera 3 minutos.
5. Apaga el fuego.
   Ahí sí puede hacerlo. **Eso es programar**: darle instrucciones claras, paso a paso, a una computadora para que haga algo por ti.

Cada vez que usas una app, escuchas música en Spotify, o abres un videojuego, alguien escribió esas instrucciones antes. Ese alguien es un programador, y tú estás a punto de convertirte en uno.
 
---

### ¿Por qué las computadoras necesitan instrucciones tan exactas?

Las computadoras son increíblemente rápidas, pero también increíblemente literales. No tienen sentido común. No interpretan. No sienten.

Si tú le dices a un amigo "espera un momento", él entiende que son unos segundos. Una computadora no sabe qué es "un momento". Necesita un número exacto.

Por eso existe la **programación**: para hablar el idioma de las máquinas de forma ordenada y precisa.
 
---

### ¿Qué es Java?

Java es uno de los **lenguajes de programación** más usados en el mundo. Un lenguaje de programación es, básicamente, el idioma que usamos para darle instrucciones a la computadora.

Así como en el mundo hay español, inglés, mandarín... en programación hay Python, JavaScript, C++, y **Java**.

Java se creó en 1995 y todavía hoy es uno de los más populares. ¿Por qué? Porque es:

- **Versátil**: sirve para hacer casi cualquier cosa.
- **Portable**: el mismo código funciona en Windows, Mac y Linux.
- **Muy usado en el trabajo real**: miles de empresas lo usan todos los días.

### ¿Por qué aprender Java siendo joven?

Hay muchos lenguajes para aprender, pero Java tiene ventajas especiales si estás empezando:

**1. Te enseña a pensar bien.** Java es un lenguaje que te obliga a ser ordenado y claro. Eso es una habilidad para toda la vida, no solo para programar.

**2. Hay trabajo garantizado.** Java es uno de los lenguajes más pedidos en ofertas de trabajo de tecnología en todo el mundo.

**3. Si aprendes Java, aprender otros lenguajes es mucho más fácil.** Es como aprender a manejar en un auto manual: después cualquier auto te parece sencillo.

**4. Puedes crear cosas reales.** Juegos, apps, páginas web, herramientas. No es solo teoría.

### Tu primer programa en Java

Este es el programa más simple que existe en Java. Se llama **"Hola Mundo"** y es la tradición de todo programador al aprender un lenguaje nuevo:

```java
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("¡Hola, mundo!");
    }
}
```

No te preocupes si no entiendes todo todavía. Lo irás entendiendo con cada lección. Por ahora, lo importante es saber que:

- `System.out.println(...)` → muestra un mensaje en pantalla.
- Todo el código vive dentro de una `class`.
- El programa empieza a ejecutarse desde `main`.
  **Pruébalo tú mismo →** [Abrir en JDoodle](https://www.jdoodle.com/online-java-compiler) y pega el código. Luego haz clic en "Run". ¡Verás tu primer mensaje!
